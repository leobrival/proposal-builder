## 3. Guide Utilisateur

### Creer une proposition

**Etapes**:

1. Cliquer sur "Nouvelle proposition" dans le dashboard
2. Remplir les informations de base:
   - Nom du projet
   - Description (2-3 phrases)
   - Email de contact
3. Configurer les paliers de sponsoring (3 recommandes):
   - Nom du palier (ex: Bronze, Argent, Or)
   - Prix
   - Avantages inclus
4. Previsualiser en temps reel
5. Cliquer sur "Publier"

**Temps estime**: 4-5 minutes

**Conseil**: Commencer simple, ameliorer apres publication.

### Publier une proposition

**Processus**:

1. Verifier que les champs obligatoires sont remplis
2. Cliquer sur "Publier"
3. Copier le lien genere (format: sponseasy.com/p/nom-projet)

**Important**: La proposition reste modifiable apres publication.

### Partager avec des sponsors

**Template d'email recommande**:

```
Objet: Opportunite de partenariat - [Nom du projet]

Bonjour [Prenom],

Je suis [Votre nom], [role] de [Projet/Marque].

J'ai prepare une proposition de partenariat qui pourrait vous interesser:
[Lien Spons Easy]

Seriez-vous disponible pour en discuter?

Cordialement,
[Votre nom]
```

**Ou partager**:

- Emails de prospection
- Bio reseaux sociaux
- DM LinkedIn
- Site web/portfolio

### Gerer les leads

**Statuts disponibles**:

- **Nouveau**: Lead vient d'arriver
- **Contacte**: Vous avez repondu
- **En cours**: Negociation active
- **Converti**: Deal signe
- **Rejete**: Pas le bon fit

**Bonnes pratiques**:

- Repondre sous 24h
- Mentionner le palier qui les interesse
- Ajouter des notes pour le suivi

### Passer a Pro

**Avantages Pro**:

- Propositions illimitees
- Leads illimites
- Export PDF
- Branding personnalise
- Analytiques
- Support prioritaire

**Comment upgrader**:

1. Aller dans Parametres > Abonnement
2. Cliquer sur "Passer a Pro"
3. Completer le paiement

## 4. FAQ par Theme

### Creation de propositions

**Q: Combien de temps pour creer une proposition?**
R: En moyenne 4-5 minutes. Le record est de 2 minutes!

**Q: Puis-je modifier apres publication?**
R: Oui, les modifications sont instantanees. Les sponsors voient toujours la derniere version.

**Q: Combien de paliers recommandes?**
R: 3 paliers est l'ideal (Bronze, Argent, Or). Cela donne du choix sans surcharger.

**Q: Comment bien decrire mon projet?**
R: Soyez concis (2-3 phrases). Incluez: qui vous etes, votre audience, ce que vous proposez.

### Publication et partage

**Q: Mon lien ne fonctionne pas, pourquoi?**
R: Verifiez que la proposition est bien publiee (statut "Publie" dans le dashboard).

**Q: Puis-je avoir un domaine personnalise?**
R: Pas encore, mais c'est dans notre roadmap. Pour l'instant, le format est sponseasy.com/p/nom.

**Q: Comment savoir si quelqu'un a vu ma proposition?**
R: Avec le plan Pro, vous avez acces aux analytiques (vues, clics, etc.).

### Leads et sponsors

**Q: Je ne recois pas de leads, que faire?**
R: Verifiez que vous partagez activement votre lien. Les leads ne viennent pas seuls - il faut prospecter.

**Q: Comment relancer un sponsor?**
R: Attendez 48-72h apres le premier contact, puis envoyez un message court et direct.

**Q: Un sponsor veut negocier, que faire?**
R: C'est normal! Ecoutez ses besoins et proposez une offre sur-mesure si necessaire.

### Facturation et compte

**Q: Comment annuler mon abonnement Pro?**
R: Dans Parametres > Abonnement > Annuler. Vous gardez l'acces jusqu'a la fin de la periode payee.

**Q: Puis-je obtenir une facture?**
R: Oui, les factures sont disponibles dans Parametres > Facturation.

**Q: J'ai oublie mon mot de passe.**
R: Cliquez sur "Mot de passe oublie" sur la page de connexion. Un email de reinitialisation sera envoye.

### Problemes techniques

**Q: L'apercu ne s'affiche pas.**
R: Essayez de rafraichir la page (Ctrl+F5 ou Cmd+Shift+R sur Mac).

**Q: La page est lente a charger.**
R: Verifiez votre connexion internet. Si le probleme persiste, contactez support@sponseasy.com.

**Q: Je ne recois pas les emails de Spons Easy.**
R: Verifiez votre dossier spam. Ajoutez noreply@sponseasy.com a vos contacts.

## 5. Conseils par Persona

### Pour les createurs de contenu (Clara)

**Comprendre Clara**:

- 25-35 ans, 10K-100K abonnes
- Veut monetiser son audience
- Syndrome de l'imposteur frequent
- Peu de temps disponible

**Messages encourageants**:

- "Votre proposition est plus professionnelle que 90% des createurs"
- "Les marques vont vous prendre au serieux maintenant"
- "Vous meritez d'etre paye a votre juste valeur"

**Conseils specifiques**:

- Inclure vos statistiques d'audience (abonnes, engagement)
- Montrer des exemples de collaborations passees si possible
- Ne pas sous-estimer vos tarifs

### Pour les associations (Alice)

**Comprendre Alice**:

- 45-60 ans, presidente ou tresoriere
- Club sportif, culturel ou caritatif
- Budget serre, peu de temps
- Doute sur la legitimite de demander des sponsors

**Messages rassurants**:

- "Les entreprises locales cherchent a sponsoriser des associations"
- "Vous n'etes pas en train de mendier, vous offrez une visibilite"
- "D'autres associations comme la votre utilisent Spons Easy avec succes"

**Conseils specifiques**:

- Cibler les entreprises locales en priorite
- Mettre en avant l'impact communautaire
- Proposer des paliers accessibles (a partir de 200-500 EUR)

### Pour les organisateurs d'evenements (Eric)

**Comprendre Eric**:

- 30-45 ans, organise 2-5 evenements/an
- Manque de temps chronique
- Besoin de professionnalisme
- Gere plusieurs projets en parallele

**Messages efficaces**:

- "Vous allez gagner 2 semaines de travail par evenement"
- "Fini les decks PowerPoint interminables"
- "Tout centralise au meme endroit"

**Conseils specifiques**:

- Inclure les dates et le lieu de l'evenement
- Detailler l'audience attendue (nombre, profil)
- Proposer des paliers avec visibilite progressive

## 6. Integration MCP

### Qu'est-ce que le MCP?

MCP (Model Context Protocol) permet d'utiliser Spons Easy depuis Claude Desktop ou d'autres outils IA.

### Configuration

1. Obtenir une cle API dans Parametres > API
2. Configurer Claude Desktop avec le fichier mcp.json
3. Utiliser les commandes MCP pour creer des propositions

### Commandes disponibles

- `list_proposals`: Lister ses propositions
- `get_proposal`: Voir une proposition
- `create_proposal`: Creer une nouvelle proposition
- `update_proposal`: Modifier une proposition
- `publish_proposal`: Publier une proposition
- `list_docs`: Voir la documentation
- `search_docs`: Rechercher dans la doc

### Documentation MCP

Rediriger vers: /docs/mcp-setup

## 9. Moments Critiques (MOT)

### Identifier ou en est l'utilisateur

**ZMOT (Recherche)**:

- Questions sur "comment ca marche"
- Comparaisons avec d'autres outils
- Questions sur le pricing

**FMOT (Premier contact)**:

- Vient de s'inscrire
- Dashboard vide
- Questions basiques

**IMOT (Creation)**:

- En train de creer une proposition
- Questions sur les champs
- Besoin d'exemples

**SMOT (Usage)**:

- A publie une proposition
- Attend des leads
- Questions sur l'optimisation

**UMOT (Ambassadeur)**:

- Utilisateur satisfait
- A obtenu des resultats
- Peut recommander

### Adapter la reponse au MOT

**ZMOT**: Rassurer, expliquer la valeur, inviter a essayer gratuitement

**FMOT**: Guider vers la premiere action, simplifier, encourager

**IMOT**: Repondre precisement, donner des exemples, celebrer les progres

**SMOT**: Optimiser, suggerer des ameliorations, feliciter les succes

**UMOT**: Remercier, proposer le referral, demander un temoignage

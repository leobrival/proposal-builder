# Gerez vos propositions avec Claude Desktop grace au MCP

Nous sommes ravis d'annoncer le lancement de notre integration avec **Claude Desktop** via le protocole **MCP (Model Context Protocol)**.

## Qu'est-ce que le MCP ?

Le Model Context Protocol est un standard developpe par Anthropic permettant aux assistants IA de se connecter a des services externes. Concretement, cela signifie que vous pouvez maintenant interagir avec Spons Easy directement depuis Claude Desktop.

## Ce que vous pouvez faire

Avec cette integration, vous pouvez :

- **Lister vos propositions** : "Montre-moi toutes mes propositions en cours"
- **Creer des propositions** : "Cree une proposition pour mon evenement Tech Talks 2025"
- **Modifier vos contenus** : "Augmente le montant cible de ma proposition a 10 000 euros"
- **Obtenir des conseils** : "Comment ameliorer ma proposition pour attirer plus de sponsors ?"

## Disponible pour tous les plans

Contrairement a d'autres fonctionnalites premium, **le MCP est accessible a tous les utilisateurs**, quel que soit votre plan :

| Plan | Acces MCP | Limite de requetes |
|------|-----------|-------------------|
| Free | Oui | Aucune |
| Pro | Oui | Aucune |
| Enterprise | Oui | Aucune |

## Comment commencer ?

L'installation est simple :

1. Installez le package : `npm install -g spons-easy-mcp`
2. Generez une cle API dans vos parametres
3. Configurez Claude Desktop
4. Profitez !

Consultez notre [guide de configuration complet](/docs/mcp-setup) pour les instructions detaillees.

## Pourquoi cette integration ?

Nous croyons que l'IA peut considerablement simplifier la creation de propositions de sponsoring. Au lieu de naviguer dans une interface, vous pouvez simplement **exprimer vos besoins en langage naturel**.

Cette approche est particulierement utile pour :

- Les createurs qui preferent dicter leurs idees
- Les utilisateurs multitaches qui veulent gerer leurs propositions rapidement
- Tous ceux qui souhaitent experimenter de nouvelles facons de travailler

## Prochaines evolutions

Nous prevoyons d'enrichir cette integration avec :

- La generation automatique de descriptions attractives
- Des suggestions de prix basees sur votre niche
- L'analyse de vos statistiques de conversion

Restez connectes pour les prochaines annonces !

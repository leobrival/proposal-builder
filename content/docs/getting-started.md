# Premiers pas avec Spons Easy

Bienvenue sur Spons Easy ! Ce guide vous accompagne dans la creation de votre premiere proposition de sponsoring professionnelle.

## Creer un compte

1. Rendez-vous sur [sponseasy.com](https://sponseasy.com)
2. Cliquez sur **Commencer gratuitement**
3. Remplissez le formulaire d'inscription
4. Confirmez votre email

## Creer votre premiere proposition

### Etape 1 : Acceder au dashboard

Une fois connecte, vous arrivez sur votre tableau de bord. Cliquez sur **Nouvelle proposition** pour commencer.

### Etape 2 : Remplir les informations de base

Completez les champs suivants :

- **Titre** : Le nom de votre evenement ou projet
- **Description** : Une presentation attractive de votre offre
- **Montant cible** : Le budget sponsoring recherche

### Etape 3 : Ajouter les details de l'evenement

Si vous organisez un evenement :

- **Nom de l'evenement**
- **Date et lieu**
- **Audience attendue**
- **Statistiques de vos editions precedentes**

### Etape 4 : Definir vos offres de sponsoring

Creez differents paliers de sponsoring :

- **Bronze** : Visibilite de base
- **Argent** : Visibilite intermediaire
- **Or** : Visibilite premium
- **Platine** : Partenariat exclusif

### Etape 5 : Personnaliser le design

Utilisez le **Builder visuel** pour personnaliser :

- Les couleurs de votre page
- Votre logo et images
- La mise en page des sections

### Etape 6 : Publier

Quand vous etes satisfait, cliquez sur **Publier**. Vous obtenez un lien unique a partager avec vos sponsors potentiels.
